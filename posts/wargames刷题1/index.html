测试一下就可以知道argv不会截断’\ ‘，access会连带空格当成文件名，而snprintf生成的/bin.cat命令会按空格隔开

int main(int argc, char *argv[])
{
    printf("argc: %d\n", argc);
    for(int i = 0; i < argc; i++) {
        printf("access: %d\t", access(argv[i], 0));
        printf("argv: %s\n", argv[i]);
    }
    return 0;
}
